## 🎯 Usage Guidelines

### Color Usage
```css
/* Positive/Profit */
color: var(--success);

/* Negative/Loss */
color: var(--danger);

/* Neutral Data */
color: var(--text-primary);

/* Labels */
color: var(--text-muted);

/* Accents */
color: var(--accent-primary);
```

### Typography
```css
/* Financial Data */
font-feature-settings: "tnum" 1;
font-variant-numeric: tabular-nums;

/* Headings */
font-weight: 800;
letter-spacing: -0.02em;

/* Labels */
text-transform: uppercase;
letter-spacing: 0.05em;
font-size: 0.75rem;
```

### Spacing
```css
/* Section gaps */
gap: var(--section-gap); /* 2rem */

/* Card padding */
padding: 1.5rem;

/* Button padding */
padding: 0 1.75rem;
height: 3rem;
```
